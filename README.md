## Run
- `python get_car_type.py`
  - 获取二手车的品牌信息
- `python get_car_id.py`
  - 获取二手车的id，需要使用二手车的品牌信息辅助查询
- `python get_car_info.py`
  - 根据id查询二手车详细信息
- `pre_process.ipynb`
  - 数据清洗，格式转换，关键词提取
- `dataset.py`
  - 数据的归一化在dataset里面实现了
